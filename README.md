# General info
nFinance is a personal accounting software inspired by commad line tool "ledger-cli"

## Technologies
Project is created using following technologies:
* JavaScript ES6
* HTML5
* CSS3
* Vue.js 3.1.2
* Vue-Router 4.0.10
* Vuex 4.0.2
* decimal.js 10.3.1

Testing is performed with following technologies:
* unit testing - jest
* e2e testing - cypress
